# рџљЂ Installation

Clone the repository

```bash
git clone https://github.com/ARYAN-S-23/Learnel---a-personalized-learning-app.git
```

Move into the project

```bash
cd Learnel---a-personalized-learning-app
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
